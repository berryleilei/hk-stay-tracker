# 在港天数 · 港漂全家打卡

港漂(及家庭受养人)出入境打卡工具。记录每次入境 / 离境,自动算出在港天数,用于**续签佐证**与**183 天税务自查**。

- **纯本地**:数据只存浏览器 `localStorage`,全程不联网、不上传。
- **全家**:主申请人 + 配偶 / 子女受养人,各自独立计算在港天数。
- **打卡 = 补打卡**:打卡时日期默认今天,往回选即补录历史(忘了打、刚开始用都能补)。
- **PWA**:可「添加到主屏幕」,离线可用,体验接近原生 App。

## 技术栈

TypeScript(strict)+ React 19 + Vite 6 + Tailwind 3 + vite-plugin-pwa。零后端。

## 天数计算口径(重要)

**入境日与离境日两端均计入在港天数**(香港入境处通行算法)。
一段「入境 A → 离境 B」的在港天数 = `B − A + 1`(含头含尾)。仍在港的开放区间 end 取今天。

各项统计定义:

| 指标 | 定义 |
|------|------|
| 累计在港总天数 | 全部在港区间去重后的总天数(同日出又进只计一次) |
| 累计离港天数 | 自首次入境至今的总跨度 − 在港天数 |
| 本次连续在港 | 在港时:最近入境日至今(含两端) |
| 滚动 12 个月在港 | 今天往前推 12 个月 + 1 天 至今,窗口内在港天数(**续签佐证**) |
| 183 税务阈值 | 滚动 12 个月 / 本公历年 在港天数,与 183 比较(可切换口径) |
| 永居进度 | 永居起算日(默认首次入境)至「满 7 年」的进度 |

### 已知 limitation

- **滚动 183 用的是「截至今天的 12 个月」窗口**,不是「历史上任意 12 个月的最大值」。作个人自查的实用近似;严格税务判定请咨询专业人士。
- 永居「通常性居住满 7 年」是法律概念(连续性、居所意图等综合判断),本工具的「7 年进度」只按日历时长粗算,**不构成永居资格判断**。

> ⚠️ 本工具仅作个人记录与自查,**不构成续签 / 税务 / 永居资格的法律意见**。请以入境处、税务局及专业顾问为准。

## 数据与隐私

- 数据仅存本机浏览器 `localStorage`,不上传任何服务器。
- 清浏览器缓存 / 换设备会丢数据 → 用首页「导出备份」存 JSON,新设备「导入恢复」。

## 为什么是 Web/PWA 不是微信小程序

微信小程序自 2023-09-01 起**强制 ICP 备案**(前置还需微信认证 300 元/年、实名 + 通管局核验),且受大陆内容监管。本工具面向在港用户、数据纯本地,做 Web/PWA **免备案、零成本、最快上线**,也更契合「数据不上传」。如将来需要微信内传播获客,核心算法(`src/lib/`)可原样复用到 Taro / 原生小程序。
